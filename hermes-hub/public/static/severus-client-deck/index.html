CLAUDE.md — The Brain's Rulebook
The AI follows rules, not vibes
Every Claude agent in Severus OS runs against a strict instruction file. This makes the AI deterministic — it behaves the same way every time, no hallucination drift.
1

Think Before Acting

State assumptions. Ask when uncertain. Push back when a simpler path exists.

2

Surgical Changes Only

Touch only what's needed. Never refactor adjacent code. Match the existing system.

3

Goal-Driven Execution

Define success criteria first. Loop until verified. Don't follow steps blindly.

4

Token Budgets Are Hard Limits

4,000 tokens per task. Surfaces the breach. Never silently overruns.
