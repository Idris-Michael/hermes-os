# Task: UGC Outreach Collateral

## Goal
Create the sales copy, intake form link, and client onboarding email so Idris
can start selling the UGC service immediately — before Phase 1 dashboard is built.

## Constraints
- No code changes required — this is copy/collateral only
- All deliverables as Markdown files under `.agents/ugc-outreach/`
- Pricing aligned with plan: Package 1 = £400–600, Package 2 = £600–800

## Deliverables

### 1. `.agents/ugc-outreach/SALES-PAGE.md`
Short-form sales page copy for:
- Headline + subheadline (problem-aware hook)
- 3 bullet benefits (outcome-focused)
- Package 1 description (£400–600, 5 variants, delivery + download)
- Package 2 description (£600–800, 5 variants + 3-platform publishing + metrics)
- Social proof placeholder (fill once first client delivers)
- CTA: "Book a call / submit brief" → link to `/ugc-intake`

### 2. `.agents/ugc-outreach/ONBOARDING-EMAIL.md`
Template email sent to a new client after intake form submission:
- Subject: "Brief received — your UGC videos will be ready in 24 hours"
- Body: Confirms brief, states delivery timeline, sets expectations on approval workflow
- Includes what they'll receive (5 MP4s + captions JSON)
- Approval link placeholder (generated by notifyUGCClient.ts)

### 3. `.agents/ugc-outreach/UPWORK-PITCH.md`
Upwork service listing copy for "AI UGC Content Package":
- Title (under 80 chars)
- Description (300–400 words)
- Packages section (3 tiers with prices)
- Portfolio section (placeholder for proof video links)
- FAQ (3 questions: turnaround, revisions, ownership)

### 4. `.agents/ugc-outreach/CLIENT-BRIEF-TEMPLATE.md`
Questionnaire template to send prospects before they submit the intake form:
- 8 questions covering: product, audience, tone, competitors, USPs, CTAs, banned words, inspiration
- Used when client needs guidance before filling the form
